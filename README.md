# Dofun - The Dodging Game

This project was created while I was learning the Unity Game Engine. It served as a practice project to understand Unity's core concepts and game development workflow.

# What I Learned

- Unity Game Engine basics
- C# scripting in Unity
- Materials and their usage
- Prefabs and why they are useful
- Trigger Colliders
- Rigidbody Physics
- Basic obstacle and trap mechanics

# About the Game

This is a simple 3D game prototype where the player must navigate through obstacles.

The game includes:
- Player movement
- Static obstacles
- Trigger zones
- Falling boulders
- Cannon traps activated by trigger areas
- Rotating spinner obstacles
